# Plan: CC-16

## Work Item
**Title:** Verbose mode

## Overview
CC-16 adds a `--verbose` flag to the CLI that prints the key inputs and outputs at each orchestrator boundary — mood, flavour profile, filtered suggestion count, and shopping tier summary. This gives enough detail to debug prompt behaviour and verify agents are receiving the right context, without logging raw prompt templates or touching agent internals.
