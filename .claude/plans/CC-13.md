# Plan: CC-13

## Work Item
**Title:** End-to-end wiring

## Overview
CC-13 completes the integration of the full pipeline so the script is runnable from Obsidian. The orchestrator already chains all agents correctly, but `main.py` only prints the result to stdout — it never writes to the vault, so nothing appears in Obsidian. This branch wires `write_suggestion()` into the main flow and adds a call to `update_preferences()` so that every run both delivers suggestions and learns from any new ratings in the cocktail log. Integration tests are added to verify the full flow from CLI entry point to vault write.
