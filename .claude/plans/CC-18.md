# Plan: CC-18

## Work Item
**Title:** Standalone log command

## Overview
CC-18 adds a `--log` CLI command so users can record a cocktail rating at any time without running the full suggestion pipeline — useful for logging after the fact or from a previous session. When `--log` is provided, `--mood` is not required and the pipeline is skipped entirely: the script validates the inputs, calls `write_to_log()`, and exits.
