# Plan: CC-17

## Work Item
**Title:** Interactive cocktail log prompt

## Overview
CC-17 adds an interactive prompt at the end of every suggestion run so users can immediately log the cocktail they made without running a separate command. After suggestions are written to the vault, the script parses the cocktail names back out of `Cocktail Suggestions.md` and asks: "Which cocktail did you make? (1/2/3, or Enter to skip)". If the user picks one, they are prompted for a rating and optional notes, and the entry is written to the log.
